
# Vesper Launcher Website

Welcome to the official website for the **Vesper Launcher** – a modern Minecraft launcher built for speed, simplicity, and the power user.

---

## 🌟 What is Vesper Launcher?

Vesper is a next-generation Minecraft launcher designed to launch your game **instantly** — without the bloat of Electron-based apps. With native mod support, seamless Microsoft authentication, and a sleek, distraction-free UI, Vesper gets you playing faster with less hassle.

Explore Vesper's features, system requirements, and guides at [launcher.devflare.de](https://launcher.devflare.de).

---

## 🚀 Features

- **Instant Launch:** No lag, no heavy frameworks. Launch directly into your Minecraft server in seconds.
- **Mod Support:** Native integrations with Modrinth and CurseForge for effortless mod installation.
- **Modern Aesthetic:** Clean, minimalist interface that puts you in control.
- **Microsoft Security:** Secure OAuth login with zero telemetry.
- **Rich Integrations:** Discord RPC and modloader presence support out-of-the-box.

## ❓ Frequently Asked Questions (FAQ)

**Q: Is Vesper open source?**  
A: Yes! Contribute or report issues at [GitHub](https://github.com/IMDevFlare/vesper-website).

**Q: What platforms are supported?**  
A: Windows 10/11 (64-bit) at launch. Linux support is planned for the public beta. macOS support may come later.

**Q: Can I migrate from other launchers?**  
A: Import tools are planned. Until then, you can manually import your instances.

**Q: Is my data safe?**  
A: Yes. Vesper uses Microsoft OAuth2 for login and never tracks or sends unnecessary data.

For more answers, see the [FAQ](https://launcher.devflare.de/#faq) on the website.
